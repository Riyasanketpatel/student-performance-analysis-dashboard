# Student Performance Analysis Dashboard

This project is an interactive dashboard designed to analyze how different student habits affect academic performance.  
It uses Exploratory Data Analysis (EDA) and machine learning models to identify patterns between lifestyle factors and exam scores.

## Project Overview

The dashboard allows users to explore relationships between variables such as study hours, attendance, screen time, stress levels, and exam scores.  
It provides visual insights and compares different machine learning models used for prediction.

## Features

- Exploratory Data Analysis (EDA)
- Data visualization using graphs
- Correlation analysis between student habits and performance
- Model comparison (e.g., Linear Regression, Random Forest)
- Interactive dashboard interface
- Dataset upload functionality

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Dashboard Components

The dashboard includes:

- Data overview

## How to Run the Project

1. Clone the repository

git clone https://github.com/Riyasanketpatel/student-performance-analysis-dashboard.git

2. Install required libraries

pip install -r requirements.txt

3. Run the application

python app.py

4. Open the dashboard in browser

http://127.0.0.1:5000/
